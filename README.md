## Requirements

- Python 3.8+
- ttkbootsrap (version tested 1.10.1)
- PIL/pillow
- nltk

## How to run

1. ```pip install ttkbootstrap```
2. ```pip install pillow```
3. ```pip install nltk```

*Intended for use on Windows operating systems*
